Khmer Handwritten char Stroke Dataset with Coordinate Labels. 
Each entry in the dataset represents a Khmer character written as a sequence of pen strokes. Each entry begins with the character label. Each stroke consists of a series of x,y coordinate pairs, ending with # stroke_end to indicate the completion of that stroke.
